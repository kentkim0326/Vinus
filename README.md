# Vinus

**The world\'s first AI-powered Web3 creator platform.**

> Anonymous. Transparent. Global.

## What is Vinus?

Vinus is a decentralized creator subscription platform on Base Network. Creators publish content and fans subscribe or purchase using USDC, USDT, or ETH. AI translates everything to 20 languages in real time.

## Features

- **MetaMask Login** — No email, no KYC. Your wallet is your identity.
- **AI Translation** — Posts, comments, DMs auto-translated to 20 languages.
- **Crypto Payments** — USDC, USDT, ETH on Base Network.
- **3-Level Referral** — On-chain, automatic, forever.
- **80/20 Revenue Split** — Creators keep 80%.
- **Dark/Light Theme** — With custom branded logos.

## Tech Stack

- Next.js 16 + TypeScript + Tailwind CSS v4
- wagmi + viem (MetaMask, Base Network)
- 20-language i18n system
- CSS variable theming
- Netlify deployment

## Development
